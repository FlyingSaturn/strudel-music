# strudel-music

A collection of music files created with [Strudel](https://strudel.cc/), a live coding music platform.

## About

This repository serves as a personal archive for Strudel music compositions and experiments. Strudel is a browser-based live coding environment for algorithmic music, inspired by TidalCycles.

## Structure

- `patterns/` - Individual pattern files and compositions
- `experiments/` - Experimental sounds and technique explorations
- `samples/` - Custom samples and audio files (if any)

## Usage

Each `.js` file contains Strudel code that can be:
1. Copied and pasted directly into [strudel.cc](https://strudel.cc/)
2. Loaded via the import feature in Strudel
3. Shared with others via links or code snippets

## Contributing

This is a personal collection, but feel free to:
- Use any patterns as inspiration for your own work
- Suggest improvements via issues
- Share your own Strudel creations
