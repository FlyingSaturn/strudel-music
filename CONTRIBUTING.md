# Contributing to strudel-music

Thank you for your interest in this collection! This is primarily a personal music repository, but contributions and suggestions are welcome.

## Adding Your Own Patterns

If you'd like to share your Strudel creations:

1. Fork this repository
2. Add your pattern files to the appropriate directory:
   - `patterns/` for complete compositions
   - `experiments/` for sketches and explorations
3. Include comments in your code explaining:
   - What the pattern does
   - Any interesting techniques used
   - Tempo and musical key (if applicable)
4. Submit a pull request with a description

## File Guidelines

- **Format**: Save files as `.js` (JavaScript)
- **Naming**: Use lowercase with hyphens (e.g., `my-pattern-name.js`)
- **Comments**: Add helpful comments to explain your code
- **Testing**: Test your pattern on [strudel.cc](https://strudel.cc/) before submitting

## Code Style

- Use clear, descriptive variable names
- Add comments for complex patterns
- Keep patterns modular when possible
- Include usage instructions for non-obvious patterns

## Questions or Issues?

Feel free to open an issue if you:
- Find a pattern that doesn't work
- Have suggestions for organization
- Want to discuss Strudel techniques
- Have questions about the code
