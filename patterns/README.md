# Patterns

This directory contains complete musical patterns and compositions created with Strudel.

## How to Use

1. Copy the contents of any `.js` file
2. Open [strudel.cc](https://strudel.cc/)
3. Paste the code into the editor
4. Press `Ctrl+Enter` (or `Cmd+Enter` on Mac) to play

## File Naming Convention

Use descriptive names that indicate the style or mood:
- `ambient-drone.js`
- `techno-beat.js`
- `melodic-experiment.js`
- `jazz-chords.js`

## Tips

- Add comments in your code to explain interesting techniques
- Include tempo and key information in comments
- Document any external samples used
